Agregar las tablas del archivo PostgreSQL -> Create_Tables.sql, luego los inserts en PostgreSQL -> Inserts.sql y los triggers, procedimientos y funciones en PostgreSQL -> Triggers.sql, Funciones.sql, Procedimientos.sql

Modificar los datos de la base de datos del archivo database.js. Poner la contraseña y el nombre de la base de datos.

Asegurarse de tener instalado node.js

Correr el programa en una nueva terminal. Se pone: node api.js (Ya que usamos Node.js)

Nos dirigimos a http://localhost:3300

Para realizar cambios, simplemente recargar la página (F5)
